🤖 JARVIS AI - Personal Assistant
A powerful, Python-based virtual assistant inspired by Iron Man's JARVIS. This assistant uses Speech Recognition, NLP (GPT Integration), and GUI Automation to handle daily tasks through voice commands.

✨ Features
    🗣️ Voice Interaction: Fully functional Text-to-Speech (pyttsx3) and Speech-to-Text (SpeechRecognition).
    🤖 GPT Powered: Integrated with OpenAI for intelligent conversations and complex queries.
    📧 Communication: Send automated WhatsApp messages and Emails via voice.
    🖥️ System Control: Open any desktop application using PyAutoGUI and system search.
    🌐 Web Automation: Search Google, YouTube, and Wikipedia instantly.
    📝 Task Management: Create, view, and delete daily "To-Do" lists with local file storage.
    🔔 System Notifications: Desktop alerts for task reminders using the plyer library.

🛠️ Built With
    Python - Core language.
    OpenAI API - For GPT-based intelligence.
    Google Speech API - For accurate voice recognition.
    PyAutoGUI - For desktop automation.
